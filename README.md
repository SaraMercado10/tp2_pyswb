# 🌍 Turismo Pro - Sitio Web Educativo

## 📌 Descripción del proyecto

**Turismo Pro** es un sitio web educativo desarrollado con HTML, CSS, Bootstrap y JavaScript que simula una agencia de viajes moderna. El proyecto incluye múltiples páginas funcionales, diseño responsive y componentes interactivos como filtros, modales, validaciones y animaciones. Además, incorpora un módulo educativo de ciberseguridad (phishing) para concientización del usuario.

## 🎯 Funcionalidades principales

### 🧭 Navegación
- Navbar responsive en todas las páginas  
- Menú activo según sección  
- Botón de modo oscuro (Dark Mode)  

### 🎨 Diseño UI
- Cards con imágenes uniformes  
- Sistema de grillas responsive  
- Uso de Bootstrap para consistencia visual  

### 🔎 Filtros dinámicos
- Filtro de blog por categorías (Playa, Montaña, Ciudad)  
- Filtrado dinámico sin recargar la página  

### 💬 Sistema de comentarios
- Agregado dinámico de comentarios en blog  
- Inserción en DOM con animación  

### ⭐ Sistema de rating
- Selección de estrellas interactiva con jQuery  

### 📊 Tabla de precios
- Comparador de destinos  
- Tooltips informativos en cada paquete  
- Destacado visual de paquetes premium  

### 📩 Formulario de contacto
- Validación en tiempo real  
- Spinner de carga simulado  
- Modal de confirmación de envío  

### 🔐 Módulo educativo de phishing
- Simulación de mensaje fraudulento  
- Identificación de señales de ataque  
- Feedback interactivo de aprendizaje  

---

## 🌙 Modo oscuro
- Activación con botón toggle  
- Cambio dinámico de íconos  
- Aplicación global mediante clase `dark`

## 🧠 Lógica JavaScript

El archivo `main.js` incluye:
- Control de modo oscuro  
- Animación de contador  
- Filtros de destinos y blog  
- Inicialización de tooltips Bootstrap  
- Sistema de rating con estrellas  
- Validación de formularios  
- Manejo de modales  
- Interacción del módulo de phishing  

---

## 🎥 AOS Animations
Se utiliza AOS para animaciones suaves al hacer scroll en secciones del blog.
